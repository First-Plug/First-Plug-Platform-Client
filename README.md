## 3. Getting Started

This project is not yet deployed, but you can try it in your local. Follow the instructions below to run it.

#### ðŸ’  INSTALLATION

1. Clone the repository

   ```sh
   git clone https://github.com/RafaMojica/First-Plug-Platform-Client.git
   ```

2. Install NPM dependencies

   ```sh
   npm install
   ```

3. Create a file in the root .env.local then use the .env.example file found in this repository as a base

   ```sh
    NEXTAUTH_URL=somedata
    NEXTAUTH_SECRET=somedata
    GOOGLE_CLIENT_ID = somedata
    GOOGLE_CLIENT_SECRET = somedata
    AZURE_AD_CLIENT_ID=somedata
    AZURE_AD_CLIENT_SECRET=somedata
    AZURE_AD_TENANT_ID=somedata
   ```

4. Run the server

   ```sh
   npm run dev
   ```
